# Dashboard de Cotización de Proveedores

Aplicación web para comparar precios entre proveedores y detectar diferencias de cotización.

## Funciones

- Carga de proveedores por producto
- Comparación automática de precios
- Cálculo de diferencias porcentuales
- Alertas para precios anómalos
- Exportación a Excel

## Tecnologías

Node.js  
HTML  
CSS  
JavaScript
